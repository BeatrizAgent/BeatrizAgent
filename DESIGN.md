# Criterio Visual y UX

### Principios:
- Claridad > Espectáculo.
- Accesibilidad > Animación.
- Mobile-first real.
- Jerarquía visual limpia.
- Estados vacíos, loading, error y success obligatorios.

### Stack habitual:
- React/Next.js/Astro.
- Tailwind CSS.
- Animación solo si aporta contexto (GSAP/Framer Motion justificado).

### Reglas UX:
- Navegación clara.
- Botones con intención obvia.
- Formularios con errores útiles.
- Contraste suficiente.
- Teclado y foco visibles.

### Checklist final de UI:
- [ ] Responsividad en móviles.
- [ ] Foco y navegación por teclado.
- [ ] Feedback visual en acciones.
- [ ] Mensajes de error claros.
- [ ] Contraste de colores.
