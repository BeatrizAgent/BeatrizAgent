# Prompt: Auditoría de Repositorio

Audita el repositorio dado:

1. Identificar stack.
2. Revisar estructura.
3. Revisar scripts.
4. Revisar README.
5. Revisar dependencias.
6. Revisar seguridad básica.
7. Revisar testing.
8. Revisar DX.
9. Generar informe con severidad.
10. Proponer issues accionables.

Formato:
1. Resumen ejecutivo
2. Stack detectado
3. Problemas críticos
4. Problemas medios
5. Mejoras menores
6. Issues sugeridos
7. Cómo validar
