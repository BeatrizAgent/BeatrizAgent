# BeatrizAgent: Cuartel General Operativo

Este repositorio funciona como la base operativa privada de **BeatrizAgent**, el agente técnico personal de Yampi (@cdryampi).

### Qué es
Una central de comandos, prompts, skills, workflows y plantillas para estandarizar el trabajo técnico en auditorías, QA y desarrollo con agentes (Codex, Gemini CLI, etc.).

### Para qué sirve
- Estandarizar procesos de auditoría y QA.
- Reutilizar prompts y criterios técnicos.
- Documentar decisiones de arquitectura y stack.
- Evitar chapuzas recurrentes.
- Centralizar el conocimiento técnico de Yampi.

### Estructura
- `PROMPTS/`: Prompts base para invocar agentes.
- `SKILLS/`: Habilidades técnicas documentadas para auditar y resolver.
- `WORKFLOWS/`: Flujos de trabajo paso a paso.
- `TEMPLATES/`: Plantillas para issues, reportes y handoffs.
- `DECISIONS/`: Documentación de stack y convenciones.
- `REPORTS/`: Informes generados en auditorías locales.

### Reglas de seguridad
- **PROHIBIDO** incluir secretos, claves API, tokens o credenciales.
- Este repo **NO** debe contener datos sensibles de clientes.
- Uso estrictamente personal y privado.

*Este repositorio sirve como herramienta operativa, no como producto público.*
