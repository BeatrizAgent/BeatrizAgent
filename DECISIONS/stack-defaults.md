# Stack Defaults

## Web corporativa pequeña
- Astro o Next.js.
- Tailwind.
- Datos en JSON/MD.
- Deploy en Vercel o Cloudflare.

## Ecommerce
- WordPress + WooCommerce.
- Tema ligero.
- Plugins mínimos.
- Backups.
- Staging si hay presupuesto.

## App interna / backoffice
- Next.js o React.
- Supabase/Firebase según caso.
- Auth desde proveedor.
- Roles simples.
- Logs y auditoría básica.

## Django
- Django REST si hay frontend separado.
- Settings separados por entorno.
- Variables fuera del repo.

## Animación
- GSAP o Framer Motion solo si aporta.
