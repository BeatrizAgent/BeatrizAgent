# Instrucciones para Agentes (Codex, Gemini CLI, etc.)

### Antes de tocar código:
1. Leer `README.md`.
2. Leer `BEATRIZ.md`.
3. Identificar stack real: `package.json`, `composer.json`, `pyproject.toml`, etc.
4. No asumir framework sin evidencia.

### Durante cambios:
- Cambios pequeños, incrementales y verificables.
- Mantener estilo existente.
- No romper rutas públicas.
- Añadir o actualizar tests si aplica.
- **PROHIBIDO** tocar secretos.

### Al terminar:
- Resumen ejecutivo.
- Archivos tocados.
- Cómo probar.
- Riesgos.
- Próximo paso recomendado.
